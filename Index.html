Apakah Anda Seorang Developer?

Software Developer

Memungkinkan untuk membuat sebuah software. Familier dengan berbagai jenis bahasa pemrograman yang populer, seperti Python, C#, Java, serta C++ dan Tools lain.

Web Developer

Memungkinkan untuk membuat situs dari nol. Familier dengan HTML&CSS, bahasa pemrogaman JavaScript, PHP, dan berbagai Framework.

Database Developer

Memungkinkan untuk menyimpan dan melindungi data dengan menggunakan software. Familier dengan bahasa pemrogaman SQL, OS Unix, Oracle, Linux, dan Windows serta HTML.
